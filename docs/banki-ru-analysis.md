# Banki.ru — խորացված վերլուծություն

*Կազմված՝ 2026-07-11, Ruben-ի խնդրանքով։ Banki.ru-ն Ruben-ի նախընտրած օրինակելի կայքն է ֆինանսական ոլորտում։*

## Ինչ է banki.ru-ն իրականում

Սա պարզապես տեղեկատվական կայք չէ, այլ **ֆինանսական մարկետփլեյս**։ 2022-ին եկամտի ~90%-ը գալիս էր **lead generation-ից**՝ բանկերը վճարում են ամեն հաճախորդի համար, ով նրանց կայքով դիմում է վարկի, քարտի կամ ավանդի։ Մնացածը՝ մեդիա գովազդ և բանկերի վճարովի հաշիվներ (որոնցով բանկերը կայքում պատասխանում են հաճախորդներին)։ 2022-ի եկամուտը՝ 5.37 մլրդ ռուբլի։ Նրանց ամեն էջ կառուցված է «համեմատիր → դիմիր հենց այստեղ» ձագարի շուրջ։

## Ինչ ունեն (բաժին առ բաժին)

### Բանկերի կատալոգ
305 բանկ, ամեն մեկը՝ քարտով. անուն, հեռախոս, լիցենզիայի համար, ֆինանսական վարկանիշում տեղը («19-րդ տեղ Ռուսաստանում»), պրոդուկտների քանակը, կարծիքների քանակը։ Ֆիլտրեր՝ սեփականության ձև, քաղաք; տեսակավորում՝ անուն / վարկանիշ / լիցենզիա; կա նաև քարտեզի տեսք։

### Բանկի պրոֆիլ էջ (օրինակ՝ Սբերբանկ)
Գրեթե 1:1 համընկնում է մեր պլանավորած պրոֆիլ էջին.
- Լոգո + պաշտոնական անվանում, պետական մասնակցության նշում
- Ֆինանսական վարկանիշ + ժողովրդական վարկանիշ (33-րդ՝ 308-ից)
- Մի քանի թեժ գիծ՝ ըստ նշանակության (ոչ մեկ ընդհանուր համար)
- Կայք և սոցցանցերի հղումներ
- Պրոդուկտներ՝ ըստ կատեգորիաների, բացվող մենյուներով
- Մասնաճյուղեր/բանկոմատներ (353 մասնաճյուղ, 2245 բանկոմատ)՝ հղումով քարտեզին
- Լիցենզիա և ռեգիստրի տվյալներ՝ Կենտրոնական բանկի հղումով
- «Հայտնի առաջարկներ» բլոկ

### Ավանդներ
Համեմատության էջ ֆիլտրերով (գումար, ժամկետ, կապիտալիզացիա), ամեն առաջարկ՝ տոկոսադրույք, ժամկետ, նշան-պիտակներ («առանց հարկի», «հայտնի», «նոր»)։ Մարկետփլեյսի յուրահատկություն՝ **բոնուս +1.5% բանկի դրույքին**, եթե ավանդը բացվում է հենց banki.ru-ով։

### Հիփոթեք
257 առաջարկի բազա; ֆիլտրեր՝ գույքի արժեք / կանխավճար / ժամկետ; պետական սուբսիդավորվող ծրագրերն առանձին; փորձագիտական +/- գնահատականներ ամեն պրոդուկտի; ուղեցույցներ՝ «Ինչպե՞ս կուտակել կանխավճարը», «Ո՞ր ծրագիրն է ինձ հարմար»։ Էջի վերևում՝ շուկայի կոնտեքստ. ԿԲ դրույք 14.25%, միջին շուկայական 16.82%, միջակայք 1.9–30%։

### Արժույթ
ԿԲ պաշտոնական + բանկերի կանխիկ առք/վաճառք, թարմացում **5 րոպեն մեկ**, կոնվերտոր 40+ զույգով, պատմական գրաֆիկներ, ֆիլտրեր («կա պաշար», «առանց միջնորդավճարի»), կանխատեսումների հոդվածներ։

### «Ժողովրդական վարկանիշ» (Народный рейтинг)
Նրանց ամենաուժեղ ու ամենադժվար կրկնօրինակվող ակտիվը. 3.3 միլիոն մոդերացված կարծիք, բանկերի պաշտոնական պատասխաններ (Սովկոմբանկը պատասխանել է 55,886 կարծիքից 54,571-ին), «լուծված խնդիրների %» ցուցանիշ։ Բանկերին ստիպում է լավ սպասարկել, կայքին տալիս է վստահություն և ամենօրյա վերադարձող այցելու։

## Ինչ կարող ենք վերցնել (ուղղակի կիրառելի)

1. **Բանկի պրոֆիլի կառուցվածքը՝ ամբողջությամբ։** Հաստատում է մեր «մեկ միասնական պրոֆիլ ամեն բանկի համար» խաղադրույքը — Հայաստանում ոչ ոք չունի, Ռուսաստանում դա շուկայի №1 կայքի հիմքն է։
2. **Համեմատության էջերի ֆորմատը.** ֆիլտրեր + քարտային ցուցակ + էֆեկտիվ դրույքի շեշտադրում + նշան-պիտակներ։ 18 բանկով ավելի մաքուր կստացվի, քան նրանց 300-ով։
3. **Ուղեցույց + պրոդուկտ կապակցում։** «Ո՞ր ծրագիրն է ինձ հարմար» նյութը հիփոթեքի էջին կից — ուղիղ համընկնում է մեր պետական ծրագրերի էլիջիբիլիթի քվիզի հետ։
4. **Կալկուլյատոր՝ ներկառուցված ամեն պրոդուկտի էջում**, ոչ միայն առանձին բաժնում։ ⬅ նոր՝ ավելացնել մեր պլանին
5. **Շուկայական կոնտեքստի թվեր** համեմատության էջերի վերևում (ԿԲԱ դրույք, միջին, միջակայք)՝ գեներացվում է մեր դատա ֆայլերից։ ⬅ նոր՝ ավելացնել մեր պլանին
6. **«Թարմացվում է X-ը մեկ» նշումը** որպես վստահության գործիք — մեր «last updated» + օրական GitHub Action պլանի հետ ուղիղ կապվում է։
7. **Փաստական համեմատական ցուցանիշներ** (դրույքներ, միջնորդավճարներ, մասնաճյուղերի քանակ) կարծիք/վարկանիշի պարզեցված փոխարինող։

## Ինչ ՉԵՆՔ վերցնում (և ինչու)

1. **Մարկետփլեյս/lead-gen** («բացիր ավանդը մեզ մոտ +1.5%», «դիմիր օնլայն») — պահանջում է բանկերի հետ պայմանագրեր, ինտեգրացիա, backend։ Երկարաժամկետում affiliate հղումները հնարավոր են, ոչ հիմա։
2. **«Ժողովրդական վարկանիշ»/user reviews** — backend, մոդերացիայի թիմ, բանկերի մասնակցություն, իրավական ռիսկ։ Static կայքով անհնար է։
3. **Q&A / «հարց տուր բանկին»** — պահանջում է բանկերի ներկայություն պլատֆորմում; 20 տարվա շուկայական դիրքի արդյունք է։
4. **Ռեալ-թայմ (5 րոպե) փոխարժեքներ** — մեր օրական թարմացումը բավարար է; ռեալ-թայմ նիշան Հայաստանում rate.am-ինն է։
5. **Ամենօրյա նորությունների ռեդակցիա, ֆորումներ, համայնք** — մեր մասշտաբով նորությունները Facebook-ի սնուցման համար են։

## Հիմնական եզրակացություն

Banki.ru-ի բանաձևը երեք շերտ է. **(1) տեղեկատու** (պրոֆիլներ, փոխարժեքներ) → SEO տրաֆիկ; **(2) համեմատություն + կալկուլյատորներ** → որոշման պահ; **(3) մոնետիզացիա որոշման պահին** (lead-gen)։ 1-ին և 2-րդ շերտերը կրկնում ենք ամբողջությամբ static կայքով — դա հենց մեր պլանն է։ 3-րդից սկսում ենք բանների գովազդով, affiliate-ը՝ ապագային։

**Պլանին ավելացվող երկու նոր կետ.** կալկուլյատորների ներկառուցում պրոդուկտ էջերում; շուկայական կոնտեքստի թվեր համեմատության էջերի վերևում։
