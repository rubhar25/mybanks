# Մրցակիցների վերլուծություն — Հայաստան + banki.ru խորացում

*Կազմված՝ 2026-07-11։ Լրացնում է [banki-ru-analysis.md](banki-ru-analysis.md)-ը։*

## 1. rate.am — փոխարժեքների առաջատարը

**Ինչ ունեն.** 19 բանկ/փոխանակման կետ, առք/վաճառք՝ կանխիկ և անկանխիկ առանձին, 25+ արժութային զույգ (ներառյալ խաչաձև՝ USD/RUB, EUR/GEL…), թարմացման ժամանիշ ամեն բանկի մոտ, կոնվերտոր, մասնաճյուղերի քարտեզ, կրիպտո բաժին, Rate.Trading պլատֆորմ, iOS/Android հավելվածներ, 3 լեզու։

**Ինչ ՉՈՒՆԵՆ (մեր հնարավորությունը).**
- Պատմական գրաֆիկներ / տրենդներ — չկա՛
- Փոխարժեքի ծանուցումներ/ալերտներ — չկա
- Բացատրական կոնտենտ (ինչ է սփրեդը, երբ է ձեռնտու փոխանակել) — չկա
- Բանկերի պրոֆիլներ, վարկեր, ավանդներ — ընդհանրապես իրենց դաշտը չէ

**Եզրակացություն.** ռեալ-թայմ փոխարժեքներում նրանց հետ չմրցել (արդեն որոշված է), բայց մեր արժույթի էջը կարող է գերազանցել նրանց հենց պատմական գրաֆիկներով + բացատրող կոնտենտով + բանկի պրոֆիլի հետ կապով (rate.am-ից բանկ սեղմելով ոչինչ չես իմանում, մեզ մոտ՝ ամբողջ պրոֆիլը)։

## 2. fininfo.am — Կենտրոնական բանկի պաշտոնական համեմատիչը

**Ինչ ունեն.** ԿԲԱ-ի կողմից շահագործվող համեմատման պորտալ. սպառողական/հիփոթեք/ավտո/ոսկու/ուսանողական վարկեր, ավանդներ, քարտեր, փոխանցումներ, պարտատոմսեր, ճամփորդական ապահովագրություն։ «Ես փնտրում եմ…» dropdown ընտրիչ, ընտրվածների հաշվիչ, կալկուլյատորներ։

**Թույլ կողմերը.**
- Միայն հայերեն, UI-ը՝ բյուրոկրատական, ոչ ոք «սիրով» չի օգտվում
- Բանկերի մասին էջեր՝ գրեթե զրո, ուղեցույցներ՝ չկան
- SEO-ն թույլ է, կոնտենտ մարկետինգ չկա

**Ինչ վերցնել.** նրանց **տվյալների ընդգրկումը** որպես ստուգիչ (ԿԲԱ-ի տվյալները վստահելի են — մեր data-collector-ի համար լավ խաչաձև ստուգման աղբյուր), և «Ես փնտրում եմ…» մուտքի պարզ մոդելը որպես UX գաղափար գլխավոր էջի համար։

## 3. afm.am — ամենամոտ ուղղակի մրցակիցը

**Ինչ ունեն.** մեկ պլատֆորմում ավանդներ, քարտեր, վարկեր, հիփոթեք (ներառյալ պետծրագրեր), թվային դրամապանակներ; կալկուլյատորներ; ԿԲԱ + բանկերի փոխարժեքներ; «Journal» կրթական բաժին; նորություններ/ակցիաներ; 16+ բանկի ցուցակ; «առանց թաքնված միջնորդավճարների» թափանցիկության շեշտադրում։

**Թույլ կողմերը (մեր տարբերակիչները).**
- **Միայն ռուսերեն** — հայերեն ու անգլերեն տարբերակ չկա։ Սա հսկայական բաց է. հայալեզու որոնումների ամբողջ տրաֆիկը անտեր է
- Բանկերի ուղղակի «պրոֆիլ» էջեր՝ թույլ (ցուցակ կա, խորը պրոֆիլ՝ ոչ)
- Կարծիքներ/վարկանիշ չկա, կոնտակտները՝ կիսադատարկ
- Հավելված չկա

**Եզրակացություն.** afm.am-ը ամենալուրջ մրցակիցն է կոնցեպտով, բայց ռուսերենով սահմանափակվելը մեզ թողնում է ամբողջ հայալեզու SEO դաշտը։ Մեր «հայերենն առաջինը» որոշումը ուղիղ հարված է նրանց ամենաթույլ տեղին։

## 4. karucapatoxic.am — ոչ ուղղակի մրցակից, բայց ուսանելի

Իրականում **նորակառույցների մարկետփլեյս** է (139 նախագիծ, կառուցապատողների պրոֆիլներ, գների համեմատում ըստ քաղաքների), ոչ թե ֆինանսական պորտալ. մեզ հետ հատվում է միայն հիփոթեքի կալկուլյատորով։

**Ինչ վերցնել.** AMD/USD գների փոխարկիչը listing-ներում, «favorites» ֆունկցիան, և գլխավորը՝ **հետագա գործընկերության գաղափարը**. մեր հիփոթեքի համեմատությունը + նրանց նորակառույցները բնական զույգ են (նրանց այցելուն վարկ է փնտրում, մերը՝ բնակարան)։

## 5. Banki.ru — խորացված գտածոներ (2-րդ անցում)

### Քարտերի համեմատության UX (ուղիղ կիրառելի մեր Cards բաժնում)
- **Արագ ֆիլտր-չիպեր** ցուցակի վերևում. «առանց տեղեկանքների», «անվճար սպասարկում», «grace 100+ օր», «բոնուսներով» — մեկ սեղմումով ֆիլտր, ոչ թե ձև
- Ամեն քարտի մոտ՝ 4 առանցքային թիվ. grace ժամկետ, տարեկան սպասարկում, բոնուս/քեշբեք, **լրիվ արժեք (ՊՍԿ)** — մեր «էֆեկտիվ դրույքի» սկզբունքի քարտային տարբերակը
- **«Համեմատել» checkbox** → կողք կողքի համեմատություն (2-3 պրոդուկտ)։ Ամբողջովին client-side է, static կայքում աշխատում է → ավելացնել մեր պլանին
- «Ավելի շատ առաջարկներ» բացվող բլոկներ

### Խմբագրական «պոդբորկաներ» (curated collections)
«Ամսվա լավագույն քարտերը», «Լավագույն ավանդները մինչև 3 ամիս» տիպի կարճ խմբագրական ընտրանիներ։ SEO ոսկի են (long-tail հարցումներ) և static կայքում հեշտ գեներացվում են դատա ֆայլերից + կարճ խմբագրական տեքստից → content-writer ագենտի բնական խնդիր։

### Նորություններ → պրոդուկտ ինտեգրումը
Ամեն հոդված ունի ներկառուցված կալկուլյատոր և հղումներ համապատասխան համեմատության էջերին. նույն պրոդուկտին տանում են բազմաթիվ կոնտենտային ճանապարհներ («ինչպես ընտրել», «լավագույնները», բառարան, նորություն)։ Սա հենց մեր interconnection մոդելի կոնտենտային կողմն է. **ուղեցույցներն ու նորությունները նույնպես ID-ներով հղվեն պրոդուկտներին**, որ գեներացվող հղումները աշխատեն նաև կոնտենտից պրոդուկտ ուղղությամբ։
- Կոնտենտի ֆորմատներ. օրվա իրադարձություններ, թեմատիկ հոդվածներ, ֆինանսական բառարան, «Սկսնակների համար» (entry-point SEO), փորձագիտական կարծիքներ։

## Պլանին ավելացվող նոր կետեր (այս վերլուծությունից)

1. **«Համեմատել» checkbox → կողք կողքի տեսք** բոլոր պրոդուկտ ցուցակներում (client-side, banki.ru-ի քարտերի բաժնից)
2. **Արագ ֆիլտր-չիպեր** համեմատության էջերի վերևում («անվճար սպասարկում», «առանց միջնորդավճարի»…)
3. **Խմբագրական ընտրանիներ** («Ամսվա լավագույն…») — գեներացվող + կարճ տեքստ, long-tail SEO
4. **Փոխարժեքի պատմական գրաֆիկներ** — ոչ ոք Հայաստանում չունի, մեր օրական GitHub Action-ը ժամանակի ընթացքում ինքն է կուտակում այդ տվյալները
5. Ուղեցույցներն ու նորությունները դատա ֆայլերում **հղվում են պրոդուկտ ID-ներին** → կոնտենտ→պրոդուկտ գեներացվող հղումներ
6. fininfo.am (ԿԲԱ) — որպես տվյալների խաչաձև ստուգման աղբյուր data-collector-ի համար
7. Հետագայի համար. karucapatoxic.am-ի հետ գործընկերության հնարավորություն (հիփոթեք ↔ նորակառույց)
